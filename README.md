# Luca Swim Times 🦈

Tracking Luca Sciamanna's meet results, events, and personal bests for
**Springfield Swim Club (SSC)** — the Tiger Sharks — in the Delco Swimming &
Diving League.

Data is scraped automatically from the league's published results every
Saturday and published to GitHub Pages. See [CLAUDE.md](./CLAUDE.md) for how
the scraper, data model, and DQ/override handling work.
